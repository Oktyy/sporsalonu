# Spor Salonu Uye Yonetim Sistemi

Bu proje Web Programlama dersi icin Node.js (Express) + MySQL kullanilarak gelistirilmistir.

## Kurulum
1. MySQL servisini baslatin.
2. `veritabani.sql` dosyasini MySQL icine calistirin.
3. `.env.example` dosyasini `.env` olarak kopyalayip veritabani bilgilerini duzenleyin.
4. Terminalde proje klasorunde `npm install` komutunu calistirin.
5. `npm start` ile uygulamayi baslatin.

## Varsayilan Yonetici Hesabi
- Eposta: `yonetim@spor.local`
- Sifre: `.env` dosyasindaki `ADMIN_SIFRE` degeri
